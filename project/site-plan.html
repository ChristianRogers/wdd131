Overview

Purpose

The purpose is to communicate which items are the best at McDonald's and Burger King. A person could navigate to this site to get a quick idea of what to order at either restaurant.

Audience

People interested in food and my opinion on them

Dynamic elements

The rankings will be stored as an object within an array. I'll use ES Modules to import the data into the main JavaScript file. Then I'll use javascript to loop through the data and generate the HTML needed to show the top 10. I'll use event listeners for buttons to filter so if a user hit the "breakfast" button, they would only see those from the top 10 that are from the breakfast menu.

Wireframes

Create two wireframes for your site. One for each page and list them here

McDonalds

The idea here is that each of the top 10 items will be generated via JavaScript from an array of objects. Each item will have an image, a title, and a short description. There will also be buttons at the top to filter by whether the item is from the breakfast or the lunch/dinner menu.

Burger King

In concept, this is familiar to the other page, but to differentiate it, instead of displaying the items all at once, it will display one at a time with "next" and "previous" buttons to cycle through them.
